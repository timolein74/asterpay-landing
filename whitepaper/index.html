1. Executive Summary

AsterPay is a crypto-native payment infrastructure designed specifically for AI applications, API services, SaaS platforms, and autonomous software agents. We provide developers with a simple way to accept global payments without the complexity of traditional payment processors.

Mission: Make it radically easier for AI and SaaS developers to monetize their products globally, with instant settlements and minimal fees.

2. The Problem

Traditional payment processors were built for e-commerce and subscription businesses. They present significant challenges for modern AI and SaaS developers:

  • High barriers to entry: Bank accounts, KYC processes, and lengthy approval times
  • Geographic restrictions: Many countries cannot access Stripe, PayPal, or similar services
  • No support for microtransactions: Per-API-call billing is impractical with traditional rails
  • Slow settlements: 2-7 day settlement times hurt cash flow
  • High fees: 2.9% + $0.30 per transaction makes small payments uneconomical
  • No agent-to-agent payments: Traditional systems don't support autonomous software agents

3. The Solution

AsterPay provides a developer-first payment infrastructure built on blockchain technology:

Non-Custodial Architecture

We generate unique HD wallet addresses for each payment. Funds are automatically swept to the merchant's payout wallet, minimizing counterparty risk. We never hold user funds.

Multi-Chain Support

Accept payments on 5 blockchain networks, allowing users to pay with whichever chain offers the lowest fees:

  • BNB Smart Chain (BSC) ‚Äî ~$0.03 gas
  • Polygon ‚Äî ~$0.01 gas
  • Base ‚Äî ~$0.01 gas

Simple Integration

4. x402 Protocol Integration

AsterPay is the first European x402 Facilitator, enabling autonomous AI agents to pay for services using the HTTP 402 Payment Required standard developed by Coinbase.

What is x402?

x402 is an open protocol that enables machine-to-machine payments over HTTP. When an AI agent requests a paid resource, the server responds with HTTP 402 and payment details. The agent autonomously completes the payment and retries the request.

AsterPay x402 Features

  • Facilitator API: Full x402 payment flow handling with 3 lines of code
  • AI Agent Wallet SDK: Pre-funded wallets with spending controls
  • Multi-chain Support: Base, Polygon, Ethereum, BNB Chain
  • Micropayments: As low as $0.001 per API call
  • Instant Settlement: Sub-second payment confirmation

Why AsterPay for x402? While Coinbase CDP, thirdweb, and PayAI offer x402 support, none provide EUR fiat off-ramps, MiCA compliance, or European data residency. AsterPay is the only x402 facilitator built for European businesses.
