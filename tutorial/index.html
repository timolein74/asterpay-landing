5 How It Works

Here's the payment flow in your app:

  1. User wants to buy credits ‚Üí Clicks "Buy Credits"
  2. Your app creates a payment ‚Üí Calls AsterPay API
  3. User pays with crypto ‚Üí USDC, USDT, or $ASTERPAY
  4. AsterPay confirms payment ‚Üí Sends webhook to your app
  5. Credits added automatically ‚Üí User can generate content

The Key Code

Creating a payment is just one API call:

// Create payment intent
const response = await fetch('https://api.asterpay.io/v1/payment_intents', {
  method: 'POST',
  headers: {
    'Authorization': `Bearer ${ASTERPAY_API_KEY}`,
    'Content-Type': 'application/json',
  },
  body: JSON.stringify({
    amount: '5.00',
    currency: 'USDC',
    chain: 'bsc',
    metadata: { user_id: '123', credits: '100' }
  }),
});

const { checkout_url } = await response.json();
// Redirect user to checkout_urlJavaScript

Handling Webhooks

When payment completes, AsterPay sends a webhook:

// Webhook handler
app.post('/webhook/asterpay', async (req, res) => {
  const { event, metadata } = req.body;
  
  if (event === 'payment.completed') {
    const { user_id, credits } = metadata;
    await addCredits(user_id, parseInt(credits));
  }
  
  res.json({ received: true });
});JavaScript
